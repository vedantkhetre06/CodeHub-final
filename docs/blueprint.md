# **App Name**: CodeHub

## Core Features:

- Secure Authentication & RBAC: User registration and login using JWT for secure sessions, implementing Role-Based Access Control (RBAC) to define permissions for Student, Teacher, and Admin roles.
- Teacher: Test & Assignment Management: Teachers can create new tests (objective and coding questions with predefined test cases and time limits) and assignments with due dates, and track student submission statuses.
- Teacher: Resource & Subject Management: Teachers can add new subjects and upload subject-specific learning resources, including PDF notes or external links, for students to access.
- AI-Assisted Coding Test Setup: Provide an AI tool to assist teachers in generating boilerplate code or initial input/output test case prompts for coding questions, streamlining the test creation process.
- Student: Interactive Learning Dashboard: A personalized dashboard for students displaying their upcoming tests, pending assignments, accessible notes, and system notifications for important updates.
- Student: Integrated Test & Assignment Environment: A dedicated, time-bound interface for students to attempt objective and coding tests using a basic code editor with predefined test case evaluation. Students can also upload assignment files and view their submission status.
- Admin: System Oversight: Administrators have comprehensive tools to manage all user accounts (add, remove, update roles) and review or delete inappropriate content within the platform.

## Style Guidelines:

- Primary color: A professional and clean medium blue (#2989CC) to convey reliability and focus, inspired by academic and technology themes.
- Background color: A very light, slightly desaturated blue-gray (#EAF0F3) providing a clean, open, and inviting canvas that is easy on the eyes.
- Accent color: A vibrant turquoise (#52DBDB) used sparingly for interactive elements and highlights, adding a modern and engaging touch. This color creates good contrast with the primary and background hues.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, tech-oriented, and structured feel suitable for application titles and section headers.
- Body font: 'Inter' (sans-serif) for its high readability and neutral, objective appearance, ideal for academic content, descriptions, and user interface text.
- Code font: 'Source Code Pro' (monospace) for clear and distinct rendering of code snippets within the coding environment or documentation.
- Utilize a consistent set of line-style, functional icons that clearly communicate their purpose and align with the professional and digital aesthetic of the platform.
- Implement distinct, responsive dashboard layouts for each role (Student, Teacher, Admin) ensuring optimal usability across various screen sizes. Maintain clean visual separation of content sections.
- Incorporate subtle and purposeful animations for user feedback, such as loading indicators, success/error notifications, and smooth transitions between dashboard views or when elements appear.
